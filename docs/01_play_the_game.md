# 01 Play The Game

Run the game from the project folder:

```bash
python run_game.py
```

Useful teacher/debug keys:

- `F1`: show help
- `F2`: show player stats
- `F3`: show collision boxes
- `F4`: show object IDs
- `F5`: restart from the current checkpoint

Run the project checker when something does not work:

```bash
python check_project.py
```
