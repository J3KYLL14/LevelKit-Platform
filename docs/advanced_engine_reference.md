# Advanced Engine Reference

The beginner helpers in `levelkit.py` create normal Python dictionaries.

Advanced students can still write dictionaries directly when they need exact control over:

- physics values such as `gravity` and `jump_power`
- custom attack dictionaries
- custom AI dictionaries
- animation dictionaries
- level files generated by the visual designer

Most students should start with:

```python
from levelkit import player, enemy, npc, healing_item, quest_item
```

Use direct dictionaries only after the helper API feels limiting.
