# 02 Build A Level

Start with the visual level builder:

```bash
python level_designer.py meadow
```

Use the designer to place solids, hazards, pickups, enemies, NPCs, exits, checkpoints, and win zones.

The designer saves to:

```text
levelkit_platform/content/levels/
```

The grid is 32x32 pixels, which works well with common pixel-art sprites.
